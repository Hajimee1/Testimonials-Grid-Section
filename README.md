# Frontend Mentor – Testimonials Grid Section

This project is a responsive testimonials section built as part of a Frontend Mentor challenge.

The goal was to create a clean, structured, and responsive layout that displays user testimonials in a visually balanced way across different screen sizes.

## 📌 Project Description

This layout displays multiple user testimonials in a structured grid system.

Each testimonial includes user information, a highlighted statement, and a detailed review.

The layout emphasizes readability, spacing, and visual hierarchy while maintaining a modern and clean design style.

---

## 🎯 Features

- Responsive design for mobile and desktop
- Structured grid-based layout
- Clear visual hierarchy between content blocks
- Consistent spacing and alignment
- Subtle hover interaction effects

---

## 🧠 Focus Areas

- Responsive layout behavior
- Visual hierarchy and readability
- Clean UI composition
- Card-based design structure
- Consistent spacing system

---

## 🧰 Built With

- HTML5
- CSS3 (Grid & Flexbox)
- Responsive design principles
- Vanilla CSS (no frameworks)
